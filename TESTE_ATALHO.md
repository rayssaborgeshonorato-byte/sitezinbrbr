# 🔐 Teste do Atalho Secreto

## Instruções:

1. **Abra a página de login:**
   ```
   http://localhost:8080/pages/login.html
   ```

2. **Abra o Console (F12 ou Ctrl+Shift+I)**

3. **No console você deve ver:**
   ```
   ✓ Sistema de Login Carregado
   Pressione ALT + SHIFT + 4 para revelar o painel secreto de login
   ✓ Sistema de atalho secreto inicializado
   ⌨️  Pressione: Alt + Shift + 4
   ```

4. **Pressione: ALT + SHIFT + 4**
   - Você verá: `🔓 PAINEL DE LOGIN REVELADO!`
   - O painel de login aparecerá na página

5. **Pressione novamente: ALT + SHIFT + 4**
   - Você verá: `🔒 Painel de login ocultado`
   - O painel desaparecerá

## Dicas:
- Se não funcionar, verifique se está digitando a tecla 4 corretamente
- Tente tanto a tecla numérica do teclado quanto a do numpad
- Abra a aba "Console" (F12) para ver mensagens de debug
